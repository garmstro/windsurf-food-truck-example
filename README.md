# Food Truck Website

---

## Suggested Approach

1. **Read `Plans/client-brief.md`** — this is your prompt context.
2. **Open Windsurf** and start a new Cascade conversation.
3. **Paste (or reference) the client brief** and ask Cascade to help you build the site.
4. **Iterate** — ask for changes, improvements, and fixes. The goal is to practice prompting, not to write code yourself.

---

## Tips for Better Prompts

- Be specific: *"Make the hero section use a dark brown background with gold text"* is better than *"make it look nicer"*
- Give context: Paste the brand colors or a section of the brief directly into your prompt
- Iterate: One change at a time tends to work better than asking for everything at once
